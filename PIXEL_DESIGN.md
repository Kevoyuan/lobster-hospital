# OpenClaw 维修医院 - 像素视觉方案
## Pixel Art Visual Design

---

## 🎮 1. 设计理念

**主题**: 复古游戏诊所 / Retro Game Clinic  
**风格**: 8-bit / 16-bit 像素艺术  
**灵感来源**: 经典街机游戏、Game Boy 时代、医疗模拟游戏

---

## 🎨 2. 配色方案

### 主色调 (Primary Colors)
| 颜色名称 | Hex | 用途 |
|---------|-----|------|
| 诊所蓝 | `#4A90D9` | 主背景、品牌色 |
| 像素绿 | `#38B764` | 成功、治愈、活力 |
| 警示红 | `#E74C3C` | 警告、损坏状态 |

### 辅助色 (Secondary Colors)
| 颜色名称 | Hex | 用途 |
|---------|-----|------|
| 像素白 | `#FFFFFF` | 高光、文字 |
| 暗黑灰 | `#2C3E50` | 轮廓、深色区域 |
| 暖橙色 | `#F39C12` | 强调、金币/积分 |
| 柔和灰 | `#BDC3C7` | 次要文字、UI边框 |

### 复古调色板 (Game Boy Palette)
```
#0F380F - 深绿 ( darkest )
#306230 - 暗绿
#8BAC0F - 中绿
#9BBC0F - 亮绿 ( lightest )
```

## 📐 7. 网格系统

- **基础网格**: 8x8 像素
- **图标尺寸**: 16x16, 24x24, 32x32
- **Logo 尺寸**: 64x64, 128x128
- **字体**: 等宽像素字体 (建议: Press Start 2P, VT323)

---

## 🎮 8. 扩展建议

### 动画帧数
- 行走循环: 4帧
- 攻击/使用工具: 3帧
- 闪烁警告: 2帧

### 音效建议 (8-bit)
- 治愈/修复: 上升音阶 beep
- 损坏: 下降/破碎音
- 点击: 短促 pop 音
